## Portfolio Website – README

This is my personal portfolio website built using **React + Vite** to showcase my skills, projects.

### Features

* Responsive personal portfolio website
* About section with career objective and background
* Technical skills section
* Projects section with GitHub links
* Contact section
* Clean and modern UI
* Smooth scrolling navigation

### Tech Stack Used

* React.js
* HTML
* CSS
* JavaScript
* Vite

### How to Run the Project

1. Open terminal inside project folder
2. Install dependencies

```
npm install
```

3. Run development server

```
npm run dev
```

4. Open in browser

```
http://localhost:5173
```
